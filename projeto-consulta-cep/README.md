# Projeto Consulta CEP

Projeto em Python para consultar endereços a partir de um CEP usando a API ViaCEP.

## Funcionalidades

- Consultar endereço pelo CEP
- Exibir logradouro, bairro, cidade e estado
- Salvar consultas em um histórico
- Listar histórico de consultas
- Tratar CEP inválido ou não encontrado

## Tecnologias utilizadas

- Python
- Requests
- API ViaCEP
- Git e GitHub
